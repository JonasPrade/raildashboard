## Alembic / Database Migrations

Use the Makefile commands for all Alembic interactions — never call `alembic` directly.

| Task | Command |
|---|---|
| Apply all pending migrations | `make migrate` |
| Create a new autogenerated revision | `make migrate-create MSG="describe the change"` |
| Run migrations inside Docker container | `make docker-migrate` |
